# 005-实践-拉取新分支并提交远程

##### 例子：希望拉取新分支，代码是 outlet 分支的代码,做完修改之后提交到远程。

1. 先进入该分支
```
  git checkout outlet
```

2. 创建分支
```
  # 创建分支并切换到了该分支上
  git checkout -b new_outlet

  # 如果不确定，可以查看下当前分支
  git branch
```

3. 做改动
```
  根据自己需求修改代码...
```

4. 提交到远程
```
  git push origin new_outlet
```
